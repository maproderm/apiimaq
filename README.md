## Instalación
### 1 Paso
**Descargar proyecto.** Correr el siguiente comando

    git@github.com:maproderm/apiimaq.git

### 2 Paso
**Crear una nueva rama** 

    git branch <nuevarama>

### 3 Paso
**Instalar Composer** 

    composer install

### 4 Paso
**Instalar npm** 

    npm install

### 4 Paso
**Correr el proyecto** 

    php artisan serve
